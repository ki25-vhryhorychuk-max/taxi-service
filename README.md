# 🚕 Taxi Premium - Сервіс замовлення таксі на Streamlit

Сучасний вебдодаток для симуляції роботи таксі-сервісу, створений на мові **Python** за допомогою фреймворку **Streamlit**. Додаток має роздільний інтерфейс для пасажирів та водіїв із автоматичним розрахунком вартості, системою промокодів та рейтингами.

---

## ✨ Основні фічі

*   **👥 Рольова модель:** Можливість реєстрації та входу як Пасажир або Водій.
*   **🚖 Розрахунок поїздки:** Вибір тарифу (Економ, Комфорт, Бізнес) та гнучкий розрахунок вартості залежно від кілометражу.
*   **🎟️ Система промокодів:** Підтримка купонів на знижку (наприклад, `TAXI2026`).
*   **👨‍✈️ Панель водія:** Водії бачать замовлення в реальному часі, можуть приймати їх та завершувати поїздки з оцінюванням клієнта.
*   **📜 Історія поїздок:** Зручні вкладки з активними замовленнями та архівом виконаних рейсів.

---

## 🛠️ Як запустити проєкт локально

### 1. Клонуйте репозиторій або скачайте архів:

git clone [https://github.com/ki25-vhryhorychuk-max/taxi-premium.git](https://github.com/ki25-vhryhorychuk-max/taxi-premium.git)
cd taxi-premium

2. Встановіть необхідні залежності:

pip install -r requirements.txt

3. Запустіть додаток Streamlit:

python -m streamlit run main.py

Структура файлів

main.py - Головний файл інтерфейсу користувача та відмальовки сторінок.

taxi_service.py - Бізнес-логіка (керування замовленнями, користувачами, тарифами).

models.py - Класи сутностей (User, Order).
